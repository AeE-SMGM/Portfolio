My Projects

Aircraft Wings

This is a CAD model of aircraft wings of a cessna 208, built as part of my aerospace engineering coursework. The model features the internal structure of the wings and their attachment points.

Aircraft Fuselage

This is a CAD model of an aircraft fuselage, built as a commissioned project. The model includes the main body structure and key components.

Wind Tunnel Stand

This is a CAD model of a wind tunnel stand, built for experimental purposes. The model is designed to hold a cylindrical test section. The dimensions are selected to accommodate the specific requirements of the experiments.

Rocket Assembly

This is a CAD model of a rocket assembly. The model includes the main body, fins, and nose cone. This was built to serve as a reference for the physical construction of the rocket.

Static Ignition Platform

This is the CAD model of our static ignition platform. This model was used as the reference for the development of our capstone project. The design was also simulated to validate its performance through analyses like stress distribution, displacement, and factor of safety.

Wing rib

This is a CAD model of a wing rib, designed for a DIY remote-controlled aircraft. The model is just one part of many components that make up the wing structure.

Mini Wind Tunnel

This is a CAD model of a mini wind tunnel, designed to serve as the reference for a course group project. The model is intended to demonstrate the principles of fluid dynamics and aerodynamics in a compact and accessible format.

Unmanned Aerial Vehicle Design

This is a CAD model of an unmanned aerial vehicle design, created as part of a university course project. The model includes the airframe, propulsion system, and control surfaces. The UAV is designed to locate illegal trespassers in military perimeters and drive them away.

Empennage

This is a CAD model of an empennage, which is the tail section of an aircraft. The model includes the vertical stabilizer and horizontal stabilizer.
